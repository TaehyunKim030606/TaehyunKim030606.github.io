# 김태현

안녕하세요.  
저는 경상국립대학교 소프트웨어공학과 2학년 학부생 김태현입니다.

저는 단순히 코드를 작성하는 개발자를 넘어, 실제 사람들의 문제를 해결하는 서비스를 만들고 싶은 개발자가 되고 싶습니다.

## 관심 분야

현재 관심을 가지고 있는 분야는 다음과 같습니다.

- 소프트웨어 개발
- 웹 서비스 개발
- 데이터베이스
- 네트워크
- 정보보안

## 현재 하고 있는 것

현재는 GitHub Pages와 Jekyll을 이용해 개발 블로그를 만들고 있으며, 앞으로 공부한 내용과 프로젝트 경험을 기록하려고 합니다다.

또한 Raspberry Pi를 활용한 네트워크 지연시간 및 패킷 손실 모니터링 시스템 연구를 진행할 예정이며, 외교부 공공데이터를 활용하여 프로그램을 만드는 과정에서 DB를 구축하기 위해 공부 중입니다.

## 나의 방향성

저는 취업도 중요한 진로라고 생각하지만, 장기적으로는 창업을 통해 직접 서비스를 만들고 운영하는 사람이 되고 싶습니다.

기술을 배우는 이유도 단순한 지식 습득이 아니라, 아이디어를 실제 제품과 서비스로 구현하기 위해서입니다.

## 앞으로의 목표

앞으로 이 블로그에는 다음 내용을 기록할 예정입니다.

- 개발 공부 기록
- 프로젝트 진행 과정
- 창업 아이디어 정리
- 연구실 활동 기록
- 실패와 개선 과정
- 성장 회고

이 블로그는 저의 공부 기록이자, 프로젝트 포트폴리오가 될 것입니다.
